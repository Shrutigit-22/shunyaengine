# ShunyaEngine

Official website for **ShunyaEngine** — done-for-you Make.com automation templates for small business owners. Near-zero investment. No code. No monthly fees.

## What It Is

A static single-page marketing website built with plain HTML5 and CSS3. No frameworks, no build tools, no dependencies.

## Stack

- HTML5 + CSS3 (single file: `index.html`)
- Google Fonts — Bebas Neue, DM Mono, Instrument Serif
- Deployed via Cloudflare Pages
